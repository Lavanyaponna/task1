An indentifier/variable:
================

Q1)What is the difference between an identifer/variable?
=======================================
1+2=3
2+1=3

It is a name inside the Python Program/Python Script/Python module.


Rule_1:
=====
          Only allow characters are AtoZ/atoz/0to9 and specail character _.


Specail characters in Python:
====================
_,space,$,@,#,.


_ is only allowed as specail character
